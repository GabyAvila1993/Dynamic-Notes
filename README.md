# Backend

## Tecnologías utilizadas

1) NestJS

2) TypeScript

3) PostgreSQL

4) TypeORM

<!-- --------------------------------------------------------------------------------------------------- -->


## Requisitos previos

1) Node.js  v22.21.0

2) npm v11.7.0

3) PostgreSQL v18.x

<!-- --------------------------------------------------------------------------------------------------- -->

## Configuración de entorno
La aplicación requiere un archivo .env en la carpeta backend con las siguientes variables:

<!-- 

DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASS=tu_contraseña
DB_NAME=notas_db

 -->
